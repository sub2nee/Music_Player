# Music Player 


### 🛠 사용 기술<br/>
Editor: VScode<br/>
Lang: HTML/CSS/JS

**배포**:vercel

https://playing-music.vercel.app/
<br>

### 💻 기능 <br/>

- 랜덤재생, 한곡만재생, 전곡재생 으로 설정 가능 
- 음악 플레이시 프로그레스바와 함께 해당음원 시간표시
- 프로그레스바로 원하는 구간선택 가능
- 컨트롤러로 음악재생,다음곡, 이전곡 선택 가능
