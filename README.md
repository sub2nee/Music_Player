# Music Player 

### 🛠 사용 기술<br/>
Editor: VScode<br/>
Lang: HTML/CSS/JS

**배포**:vercel

<br>

### 💻 기능 <br/>

- 랜덤재생, 한곡만재생, 전곡재생 으로 설정 가능 
- 플레이리스트 클릭시 해당 곡 재생
- 음악 플레이시 프로그레스바와 함께 해당음원 시간표시
